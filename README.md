
# Super DTH MVP

## Quick Start

```bash
git clone <repo>
cd super-dth
npm install
cp .env.example .env   # insert Razorpay keys
npm start
```

## Tech
- Node.js + Express
- EJS templates
- SQLite (file db)
- Razorpay (payments)
- PDFKit (proposal PDF)

## Deployment
Push to GitHub and connect to Render.com or Railway.app. Default build command is `npm install`, start command `npm start`.
